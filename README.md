# Chatbot AI

This project implements a chatbot using the OpenAI API, built with Next.js and integrated with the Vercel AI SDK. The application enables real-time interactions with OpenAI's language model, providing a smooth and modern conversational experience.

## 🚀 Features

- **Dark Mode**: Interface with dark theme support for a better visual experience.
- **OpenAI Integration**: Connects to the OpenAI API for natural language processing.
- **Vercel AI SDK**: Utilizes Vercel’s AI SDK for performance and scalability.

## 🧰 Technologies Used

- **Frontend**: Next.js, Tailwind CSS, Shadcn/UI
- **Backend**: OpenAI API
- **Hosting**: Vercel
- **Editor**: Visual Studio Code
- **Linter**: ESLint with custom configuration

## 📦 Installation & Local Development

1. Clone the repository:

```bash
   git clone https://github.com/caiovellani/chatbot-ai.git
   cd chatbot-ai
```

2. Install dependencies:
```bash
npm install
```

3. Create a .env.local file in the root directory and add your OpenAI API key:
```bash
OPENAI_API_KEY=your-openai-api-key
```

4. Run the development server:
```bash
npm run dev
```
## 🚀 Deployment
The recommended platform for deployment is Vercel, which provides native support for Next.js and seamless GitHub integration.
